## Installation

1 - Cloner le repo

2 - Aller dans le dossier "docker" avec votre terminal.

3 - Lancer la commande suivante:
```
docker-compose up
```

Dans votre navigateur vous pouvez ensuite aller sur l'adresse
```
localhost:3000
```
